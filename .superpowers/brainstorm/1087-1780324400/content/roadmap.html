🧩 冷战大战略游戏 · 分阶段路线图

将项目拆解为 11 个可独立交付的 Phase，每个 Phase 聚焦一个子系统

📍 Phase 1 — 地基：项目骨架 + 数据层

  • Next.js + TypeScript 项目脚手架
  • 核心数据模型 (Country, GameState, TechTree)
  • 游戏状态管理 (Zustand / Jotai)
  • 基础回合制循环框架
  • 世界地图 GeoJSON 数据加载

📍 Phase 2 — 地图引擎：D3.js 动态地理渲染

  • D3.js + SVG 世界地图投影渲染
  • 国家点击交互 + 高亮选中
  • 国家属性面板联动
  • Era I 羊皮纸风格底图 (CSS + SVG filter)
  • 基础 CSS 主题系统 (为 Era II/III 预留接口)

📍 Phase 3 — GUI 外壳：三栏布局 + 交互

  • 左侧面板：CWS 排行榜 Top 5 + 末日时钟
  • 右侧面板：国家情报 + 行动指令输入
  • 底部：新闻滚动条 (AI 事件输出)
  • 响应式三栏布局 + 可折叠侧栏
  • 时代 I 打字机风格 UI 字体和装饰

📍 Phase 4 — 游戏逻辑：CWS 冷战分引擎

  • CWS 计算公式 (外交/军援/事件影响)
  • 区域级 + 国家级双层 CWS 体系
  • 85 分阈值危机触发器
  • 资源/军事/影响力基础数值模拟
  • 回合结算管线 (End Turn → Calculate → Resolve)

📍 Phase 5 — 科技树系统

  • 科技树数据结构 (DAG 有向无环图)
  • 时代锁定机制 + 提前研发惩罚公式
  • 美苏不对称科技树 (各自独占分支)
  • 科技研发 UI (右侧面板内嵌)
  • 科技对 CWS/军事/经济的影响绑定

📍 Phase 6 — LLM AI 裁判系统

  • LLM API 集成层 (OpenAI / Anthropic)
  • Prompt 工程：事件生成 + 行动判定
  • CWS 作为核心权重的 prompt 注入
  • AI 生成新闻事件的格式化输出
  • 上下文记忆：历史大事记表供给 LLM

📍 Phase 7 — 动态领土变更

  • GeoJSON 数据实时修改 (合并/分裂/吞并)
  • 前端地图即时重绘渲染
  • 领土变更动画过渡
  • 边界冲突可视化

📍 Phase 8 — 联机对战：Cloudflare 实时同步

  • Cloudflare Durable Objects / WebSockets
  • 回合同步协议 (Player A 结束 → Player B 收到)
  • 房间/匹配系统
  • 断线重连
  • 双人状态一致性保障

📍 Phase 9 — Era II 视觉：CRT 雷达指挥室

  • 黑底 + CRT 扫描线着色器效果
  • 磷光绿/青色发光国界线
  • 经纬度网格覆盖层
  • 雷达扫描动画
  • 军用终端风格 UI

📍 Phase 10 — Era III 视觉：赛博全息投影

  • 六边形蜂巢网格叠加层
  • 霓虹色 (青/洋红) 高对比边界
  • 光柱军事部署可视化
  • Glassmorphism UI
  • 极简扁平设计语言

📍 Phase 11 — 打磨与平衡

  • 音效设计
  • 动效打磨
  • 游戏平衡调优
  • 边缘情况处理
  • 性能优化 (地图渲染)
  • 响应式 / 无障碍
