🏗️ 渲染管线 (Layer Stack)

从底到顶共 6 层，时代切换仅影响 Layer 0-3 的样式

5交互层 — 点击/悬停/highlight 国家 (pointer-events) hover tooltip + 选中高亮 + 国家详情弹窗
4标注层 — 国家名称 + 战略要地图标 // 文字标签, 首都标记, 军事基地图标
3特效覆盖层 — CWS 热力晕染 / 雷达扫描 / 蜂巢网格 // 时代特有的全局面罩叠加
2国家层 — GeoJSON 多边形 + 国界线 // D3 path() 渲染, fill + stroke, 可变粗细颜色
1底图装饰层 — 经纬网 / 羊皮纸纹理 / CRT 扫描线 // era-defs 中的 SVG pattern/filter
0背景层 — 海洋/底色 // era 背景色 (泛黄/纯黑/暗紫)
🔒 不变层：Layer 2(国家多边形) + Layer 4(标注) + Layer 5(交互) — 跨时代共享
🎨 可变层：Layer 0(底色) + Layer 1(装饰) + Layer 3(特效) — 随时代切换

🧬 D3.js + React 集成模式

// MapRenderer.tsx — 核心地图渲染器

const MapRenderer = () => {
  const svgRef = useRef<SVGSVGElement>(null)
  const { era, countries, selectedCountry, onSelectCountry } = useGameStore()

  // 投影：根据窗口自适应
  const projection = useMemo(() =>
    geoMercator()
      .fitSize([width, height], worldGeoJSON)
      .center([20, 30])  // 冷战焦点居中
  , [width])

  const pathGen = useMemo(() => geoPath().projection(projection), [projection])

  // 只在地理数据变化时重新生成路径
  const countryPaths = useMemo(() => {
    return countries.map(c => ({
      ...c,
      path: pathGen(c.territoryPolygons)
    }))
  }, [countries, pathGen])

  return (
    <svg ref={svgRef} className={`map-${era}`}
         viewBox={`0 0 ${width} ${height}`}>

      {/* Layer 0-1: 背景+装饰 — 通过 CSS class 切换 */}
      <defs>
        <EraFilters era={era} />
        <EraPatterns era={era} />
      </defs>
      <Ocean era={era} />
      <EraOverlay era={era} />

      {/* Layer 2: 国家多边形 — 用 memo 避免重绘 */}
      <g className="countries">
        {countryPaths.map(c =>
          <CountryShape key={c.id}
            path={c.path}
            country={c}
            isSelected={selectedCountry === c.id}
            onClick={() => onSelectCountry(c.id)}
            era={era}
          />
        )}
      </g>

      {/* Layer 3: 特效覆盖层 */}
      <g className="era-effects">
        {era === 'POST_WW2'  && <BloodStains countries={countries} />}
        {era === 'IRON_CURTAIN' && <RadarSweep />}
        {era === 'INFO_AGE' && <HexGrid />}
      </g>

      {/* Layer 4: 标注 */}
      <Labels countries={countries} projection={projection} era={era} />

      {/* Layer 5: 交互 */}
      <InteractionLayer />
    </svg>
  )
}

// 路径略微抖动实现手绘效果
function handDrawPath(pathStr: string) {
  return pathStr.replace(
    /([\d.]+)/g,
    n => (parseFloat(n) +
      (Math.random()-0.5) * 1.5).toFixed(1)
  )
}

    • ⚡ 性能优化策略

    🎯 核心策略
  • useMemo 缓存 pathGen 和 countryPaths
  • React.memo 包裹 CountryShape 组件
  • 只有选中国家变化才重绘高亮层
  • 时代切换只换 CSS class + filter，不重建 SVG
    • ⚙️ 技术指标
  • ~40 个国家多边形 → 约 3000 个 SVG path 节点
  • 预估首屏加载 <500ms
  • 时代切换 <50ms (纯 CSS)
  • 国家点击响应 <16ms (60fps)
